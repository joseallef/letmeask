## Link
- https://letmeask-aulas-41e93.web.app/


**Clone o repositório**

```bach
  git clone https://github.com/joseallef/letmeask.git
```
```bach
  # Entre na pasta do projeto
    cd letmeask
  
  # Instale as dependencias
    npm install
  # Ou
    yarn
    
  # Rode a aplicação
    yarn dev
    
  # O servidor iniciará na porta 3000 - acesse http://localhost:3000
```
